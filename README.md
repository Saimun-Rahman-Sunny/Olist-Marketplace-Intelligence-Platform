# Olist Marketplace Intelligence Platform

## Overview

This project analyzes 100,000+ real-world e-commerce orders from Olist, one of Brazil's largest online marketplaces.

The objective is to transform raw transactional data into actionable business insights using Excel, SQL, and Power BI.

---

## Business Problem

Marketplace businesses generate massive amounts of data every day.

This project aims to answer key business questions:

* Which customers generate the most value?
* Which sellers perform best?
* What causes delivery delays?
* Which product categories drive revenue?
* How can customer satisfaction be improved?
